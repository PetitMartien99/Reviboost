À propos

Synapse est un projet que j'ai commencé un dimanche après-midi quand je m'ennuyais. Près de 200 heures de codage plus tard, c'est le site que vous consultez.
Je travaille encore dessus dès que j'ai le temps ; de nouvelles fonctionalités arrivent régulièrement.
Et, petit rappel : ce site reste un petit (enfin plus si petit...) projet extrascolaire, pas une entreprise professionnelle. Les bugs arrivent, tout n'est pas parfait, mais je fais ce que je peux.
Merci à vous d'utiliser mon site : c'était un rêve d'enfant !

Aide

En résumé :
  1. Tu bosses sur l'Accueil.
  2. Tu stockes dans ton Compte.
  3. Tu récupères tes archives quand tu changes de matière.

C'est ton "établi" : là où tu crées, tu modifies et tu révises tes leçons en direct.
  • Le "Cours" : tes données actuelles (elles restent même si tu quittes la page).
  • Créer une leçon : choisis un titre et un type (définitions, dates et égalités ou verbes irréguliers).
  • Ajouter :
    • Verbes : ajoute des lignes au tableau
    • Autres :
      • Définition : concept → explication
      • Date : année → événement
      • Égalité : A → B
  • S’interroger : choisis une leçon + nombre de questions.
    Modes :
    • Non-verbes : autovalidation (conseillé), QCM ou écrit
    • Verbes : aléatoire ou colonne choisie
C'est ton "archiveur" : tu y sauvegardes tes cours sur le cloud pour ne jamais les perdre.
  • Avec un compte :
    • Une session : c'est un dossier permanent (ex : SVT, Maths).
    • Créer une session : crée un dossier vide.
    • Ajouter le cours actuel : envoie le contenu de l'accueil dans une session (met à jour si elle existe déjà).
    • Importer une session : charge une session et remplace ton cours actuel (Attention, ça écrase l'accueil, pense à sauvegarder avant).

  • Disponible pour tous : voir des sessions (programmes, verbes irréguliers…) sans avoir à tout retaper.
    Attention : les verbes irréguliers remplaceront également dans le cours dans l'accueil.
